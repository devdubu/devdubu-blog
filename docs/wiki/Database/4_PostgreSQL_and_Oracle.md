# PostgreSQL and Oracle

## 1. Oracle Database

### Data Types
- **CHAR(n)**: 고정 길이 문자열. 길이가 남으면 공백으로 채웁니다.
- **VARCHAR2(n)**: 가변 길이 문자열. 실제 데이터 길이만큼만 저장됩니다. Oracle에서는 `VARCHAR` 대신 `VARCHAR2`를 권장합니다.

### SQL Types
Oracle SQL의 분류는 표준 SQL과 유사합니다.

| 분류 | 명령어 | 설명 |
| :--- | :--- | :--- |
| **DDL** | `CREATE`, `DROP`, `ALTER` | 객체(테이블, 뷰 등) 구조 정의 |
| **DML** | `INSERT`, `UPDATE`, `DELETE` | 데이터 조작 |
| **DCL** | `GRANT`, `REVOKE` | 권한 제어 |
