# 정보 단위
컴퓨터는 0과 1 밖에 이해하지 못한다.
0,1을 나타내는 가장 작은 정보 단위를 bit 라고 합니다.

비트는 0,1 두가지 정보를 표현할 수 있습니다.

n개의 전구로 표현할 수 있는 상태는 2^n 입니다.
하지만, 우리가 사용하는 모든 프로그램의 크기를 말할 때 이를 2^n 말하진 않습니다.

아래의 단위로 우리는 흔히 말하고 있습니다.

| 단위    | 설명        |
| ----- | --------- |
| 1byte | 8bit      |
| 1kB   | 1,000byte |
| 1MB   | 1,000 kB  |
| 1GB   | 1,000 MB  |
| 1TB   | 1,000 GB  |

:::tip Word?
- 중요한 정보 단위 중 워드라는 단위도 있습니다.
- 워드(word)란 CPU가 한 번에 처리할 수 있는 데이터 크기를 의미합니다.
- 만약 CPU 가 한번에 16비트를 처리 할 수 있다면, 1워드는 16비트가 되고, 
- 한 번에 32비트를 처리할수 있다면 1워드는 32비트가 됩니다.
- 이렇게 정의된 워더의 절반 크기를 **half word**, 1배 크기를 full word, 2배 크기를 double word라고 부릅니다.
- 워드 크기가 큰 CPU 는 한번에 처리할 수 있는 데이터가 많다.
- 워드 크기는 CPU마다 다르지만, 현대 컴퓨터는 대부분은 **64비트** 입니다.


:::

# 이진법
0,1 만으로 숫자를 표현하는 방법을 알아보겠습니다.

:::note 이진법?
- 수학에서 0과 1만으로 모든 숫자를 표현하는 방법을 이진법이라고 합니다.

:::

이진법을 이용하면 1보다 큰 수도 0과 1만으로 표현할 수 있습니다.
숫자가 1을 넘어가는 시점에 자리 올림을 하면 됩니다.
