# 고급 언어와 저급 언어
우리가 프로그램을 만들 때 사용하는 프로그래밍 언어는 컴퓨터가 이해하는 언어가 아닌 사람이 이해하고 작성하기 쉽게 만들어진 언어 입니다.

컴퓨터는 이 언어를 이해하지 못합니다.
:::tip 고급 언어
- 사람을 위한 언어

:::

:::tip 저급 언어
- 컴퓨터가 직접 이해하고 실행할 수 있는 언어

:::

## 기계어
컴퓨터는 0과 1로 이루어진 이 기계어를 이해하고 실행합니다.
![Pasted-image-20240806093454.png](/img/이미지 창고/Pasted-image-20240806093454.png)
다만, 위 그림처럼 기계어를 이진수로 나열하면 너무 길어지므로 아래와 같이 16진수로 표현합니다.
![Pasted-image-20240806093526.png](/img/이미지 창고/Pasted-image-20240806093526.png)
기계어는 오로지 컴퓨터만을 위해 만들어진 언어이기에, 사람이 읽으면 그 의미를 이해하기 힘듭니다.
그래서 등장한 저급 언어가 어셈블리어 입니다.

:::note 어셈블리어
- 기계어를 읽기 편한 형태로 번역할 언어

:::
