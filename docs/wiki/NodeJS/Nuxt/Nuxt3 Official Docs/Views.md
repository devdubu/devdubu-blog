## `app.vue`
Nuxt의 entrypoint에 해당하는 파일이다.
모든 라우터는 해당 파일을 무조건적으로 거치게 되어있다.
![Pasted-image-20240823082456.png](/img/이미지 창고/Pasted-image-20240823082456.png)
```vue
<template>
  <div>
   <h1>Welcome to the homepage</h1>
  </div>
</template>
```

:::success Vue에 친숙하다면, main.js가 어디에 위치하는지 궁금할 것이다.
- `main.js`는 Nuxt 백그라운드에서 돌아가고 있다.

:::
