:::note ServerLess란?
- 서버리스를 사용하게 된다면, 서버를 관리할 필요가 없어집니다.
- 단순히 코드를 배포하기만 하면 됩니다.
- 초기의 서버리스는 서비스로의 기능 즉, FaaS를 의미 했습니다.
- 즉, 서버리스는 서버가 없는 것이 아닌 서버가 보이지 않거나, 서버를 프로비전 하지 않는 것을 의미합니다.

:::
