# Flutter 개발 환경 셋팅
## Mac OS
- 우선 flutter를 진행하기 전에 flutter SDK를 설치해야한다.
- [MacOS flutter SDK](https://docs.flutter.dev/development/tools/sdk/releases?tab=macos)
- ![스크린샷-2023-01-17-오전-9.57.31.png](/img/이미지 창고/스크린샷-2023-01-17-오전-9.57.31.png)
- 원하는 디렉토리에 해당 SDK의 압출을 풀어서 넣어주면 된다.(만약 m1맥이라면, arm 버전을 다운 받으면 된다.)
- ![스크린샷-2023-01-17-오전-9.56.46.png](/img/이미지 창고/스크린샷-2023-01-17-오전-9.56.46.png)
- Flutter를 압축을 풀게 되면 위와 같은 디렉토리로 풀리게 됩니다.
