# BackLink 정책

백링크는 화살표와 연관 되어 있는 부분들을 띄워주기에, 해당 부분에 대해서는 연관성 있게 뿌려주어야한다.

링크 건 페이지 -> 링크 당사자

이런식으로 링크가 걸리기에
특히 캘린더 쪽의 Trouble Shooting 쪽과 그 이외에 것에서 BackLink를 거는 것을 습관화 하자

BackLink를 걸게 없던 페이지라면, 방향성을 보여주면 되므로, 이때는 다음 chatper로 링크를 걸자
