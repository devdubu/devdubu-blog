## 1번 문제 HGrid 단독으로 사용 불가

HGrid를 현재 `<HGrid :conf='gridconfig'></HGrid>` 방식으로 사용하고 싶지만, Grid 자체적으로

```tsx
...
const page = hui.getPage(paramProps.pcd)
...
```

store 자체적으로 pcd를 찾아서 활용하기에, 단순한 방법으로는 구현이 불가능 합니다.

store에 수동으로 store에서 pcd를 등록 해야 하지만, HGrid를 컴포넌트 형식으로 출력이 가능합니다.

등록하지 않으면, pcd를 찾는 내부 core로직에서 pcd가 없다고 분기 처리를 합니다.

해당 부분은 따로 작업 이슈를 등록해야 할 듯
