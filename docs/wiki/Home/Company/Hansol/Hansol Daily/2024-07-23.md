# RealGrid 이슈 - CodeBox2
## CodeBox 작업

- button을 두개로 차례대로 나누진 못해도, 버튼을 인위적으로 두 개를 만들 수는 있다.
- callback 함수를 실행하기 위해서는 doubleclick 처럼 grid 내부에서 함수가 있어야지 매개변수 전달이 가능해 보인다.
- code box에서 사용해야 할 주요 변수들은 아래와 같다.
    - field값 - 해당 부분은 props로 받는 name과 codeName으로 커버 가능 - field
    - **click한 cell 의 index 값 - index**
    - **popup 창에서 선택한 값 - value**
