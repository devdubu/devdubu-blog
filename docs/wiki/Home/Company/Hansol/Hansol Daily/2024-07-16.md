# Date Picker

월만 선택 가능하게 하려면, 우선 realgrid 쪽에서는 지원하지 않기 때문에, bootstrap date-picker를 이용을 해야 한다.

우선 패스

## 목표

- yyyy * MM * dd
- yyyy * MM
- yyyy
- MM
- dd

등 다양한 형태로 해당 포멧이 올 것이다.

해당 포멧으로 올 시에 알고리즘 적으로 어떻게 해결해야 할지 고민 해보기

y는 무조건 4자리 이며, M은 2자리 d도 2자리 이다.

1. y, M, d의 존재 여부를 체크
    1. `y = ([0-9]{4})`
    2. `M = ([0-9]{2})`
    3. `d = ([0-9]{2})`
2. y,M,d 이외의 문자열이 존재하는지 체크
    4. 각 문자열이 어느 위치에 있는지 체크
    5. 특수 경우이긴 하나, `-` , `/` 이 올 가능성도 생각
3. 만약 해당 inputFormat이 `yMd 갯수 -1 ≠ 특수문자 갯수` 라면 에러 뱉기
4. inputFormat은 `([0-9]{4})([0-9]{2})([0-9]{2})` 을 맞춰야 한다.
