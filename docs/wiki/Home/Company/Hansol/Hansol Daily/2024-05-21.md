# ChartJS 검토

- ChartJS 개발은 완료

[📈 vue-chartjs](https://vue-chartjs.org/)

## 문제점

- type을 커스텀해서 사용하게 된다면, 타입 충돌이 납니다.
- chartjs가 가지는 자체적인 type을 사용해야지만, 해당 타입 충돌을 막을 수 있는데
- 사실상 불가능하여서, any로 타입을 지정해야지 에러가 나지 않습니다.
- typescript의 type을 사용하지 못한다는 단점
- realchart와는 다르게, datasets의 데이터가 자유롭지 못하다는 단점을 가지고 있다 보니, 확실히 자유도는 realchart보다는 떨어지지만,
- 무료이고, 간단한 차트 정도는 커버 할 수 있을 정도 입니다.
- 메인으로 검토하기엔 무리가 있어보입니다
