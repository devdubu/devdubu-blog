# 성과 평가

올해 직무 변경하고, 처음으로 표준 웹 프레임워크 v3 전환을 맡았습니다.

중간에 목표에 대한 변경 점이 존재하긴 했지만, 이번 년도 동안에는 Grid 부분을 맡았습니다

버전 3로 올라가면서 기존에 사용하던 것과 완전히 다른 기술로 선행 해야 해 어려움이 있었지만,

공식문서의 파트 원들과의 토의로 빠르게 적응해 나아갔습니다.

프로젝트에서 Grid 쪽의 요구 사항이 많고, Grid 컨트롤이 대부분이기에, 릴리즈 목표치 그 이상을 기능 개발을 하려고 노력 했으며,

실제로도 릴리즈 현재까지 릴리즈 기준으로 목표치를 95% 완료 하였고, 릴리즈 목표 이외에 추가적으로 프로젝트에서 요구하는 기능까지 개발하였습니다.

Grid 이외에도, 차트와 토스트 부근을 추가적으로 개발 진행하였고, 3월까지는 셀렉트 박스, 인풋 박스 등을 추가 개발하였습니다.

테스트 툴 검토 중, ui와 유닛 테스트인 vitest와 e2e 테스트 툴인 msw를 검토 및 적용하는 개발을 진행하였습니다.

버전 3의 성능 개선에 대한 방식 또한 개발하고 있으며,

배포에 대한 방식은 사내 문제가 되는 버전 관리 부터 연구소의 배포 프로세스까지 50% 정도 개발하였고, 현재도 진행 중 입니다.
