# Validation 회의

## Grid Validation

- 우선 Grid Validation이 되려면 아래의 함수를 사용합니다.
- `validateColumn()` , `validationFail()`
- 해당 위의 함수를 통해서 grid 자체적으로 validation을 진행합니다.

우선 `ValidateColumn()` 이 Validation 중 가장 메인이 되는 로직입니다.

즉, validation 에 대한 이벤트를 호출하면 무조건적으로 `ValiddateColumn()` 을 거치게 되어있습니다.

우선 현재 `ValidateColumn()` 은 required 가 붙어있는 컬럼만 required가 적용되며, 해당 로직은 `isValid()` api와 validation 알림에서도 동일하게 적용됩니다.
