# HGrid

Grid의 기능 리스트

## edit

:::bug Edit은 수정 되는 컬럼과 되지 않는 컬럼을 구분해야 함



:::

- **수정**이 **가능** 컬럼은 `editable: true` 같은 구분하는 **옵션**을 따로 빼야 함
- 하지만, render에 따라서 이를 달리 적용 해야 한다.
    - 만약 checkbox renderer인데, type이 text라면, 이는 `editable: false` 로 해야지 더블 클릭 시 Text 수정으로 안 넘어 간다.
        - 이 checkbox의 editable은 따로 설정 해야 한다.
- onClick시에 바로 edit mode로 들어가는 행위 - 내부 로직

## checkbox

- 컬럼에 전체 선택
- 체크 박스 크기 조정

## selectbox

- 텍스트 입력 시 유사 데이터가 나와야 함

## DoubleClick Event

- doubleClick Evnet 정보가 없다면, doubleclick을 해도 반응이 없어야 함

## 현재 configuration

```tsx
columns: [
	{ text: '코드CD', field: 'CODE_CD', width: 100 },
  { text: '코드명', field: 'CODE_NM', width: 150 },
  { text: '언어', field: 'LANG', width: 80 },
  { text: '순서', field: 'CODE_ORD', width: 50 },
  { text: '사용', field: 'USE_FL', width: 100 },
  { text: '기타필드1', field: 'ETC_FIELD01', width: 150 },
  { text: '최종수정시간', field: 'UPDATE_DT', width: 150 },
],
```
