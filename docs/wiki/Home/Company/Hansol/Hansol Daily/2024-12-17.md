# CBtn의 문제점

우선 CBtn의 문제점은 VBtn 내부에 어떤 Content를 배치하게 된다면, 해당 `v-btn__content` 내부에 배치하게 된다.

문제는 `icon` 라는 property를 사용하는 순간부터 해당 Template을 강제로 적용하는 느낌이 든다.

`icon` property로 작성한 부분은 v-btn__content 부분에 배치가 되므로 이 부분을 제어 해야할 듯 해보인다.
