## GitLab

- gitlab 로그 확인 경로
```bash
cd /var/log/gitlab/gitlab-rails
```
- gz 압축 풀기
```bash
gzip -d application.log.2.gz
```

### gitlab group과 repository 저장 위치
- gitlab은 group은 dir로 표기 되고, repository는 해쉬 값 경로로 들어간다.
```bash
cd /var/opt/gitlab/git-data
cd repository
cd @hased

```
