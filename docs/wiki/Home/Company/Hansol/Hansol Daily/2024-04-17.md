# Component 규칙

defualt로 값을 설정하는 것과 type을 하는 것 즉

```tsx
interface Ex {
	value1: string //무조건 값이 들어와야하는 값
	value2?: string // optional 하게 처리하는 값
	value3? : string // optional 하게 처리하는 값
}

// 필수 값만 고려할 것인지 - 개인적인 생각
conf : { 
	type: Object as PropType<Ex>,
	default: ()=>{
		value1: ''
}

// 전체 값들을 모두 default 값을 넣어줄 것인지
conf : { 
	type: Object as PropType<Ex>,
	default: ()=>{
		value1: '',
		value2: '',
		value3: ''
}
```
