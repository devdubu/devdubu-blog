# Date Form

해당 Date 에 대한 form은 아래의 사이트와 같이 매우 다양하다.

입력 역시도 매우 다양할 수 있다.

[날짜 및 시간 형식](https://www.ibm.com/docs/ko/cmofm/10.5.0?topic=reference-date-time-formats)

결국은 `string.lenght` 로 해당 부분을 고려 하는 것은 한계가 있다고 판단이 된다.

아무리 저 정도의 모든 format을 고려하려면 `lenght` 보다는 어쩌피 형식에서

알파벳은 `y` , `M` , `d` 형식으로 되어있고, 규칙을 조금 따라가다보면 아래의 방법대로 진행해보려고 한다.

## 효율적으로 DateForm을 고르는 방법

### 특정 문자 갯 수

```jsx
var str = 'HTML,CSS,JavaScript';
var count = str.match(/,/g).filter(function(item) { return item !== ''; }).length;

// 결과 : 2
```

**특수 문자를 따로 출력하는 코드**

```jsx
function extractSpecialCharacters(inputString) {
    // 숫자와 알파벳을 제외한 모든 문자를 매칭하는 정규식
    const specialCharPattern = /[^a-zA-Z0-9\\s]/g;

    // 정규식을 이용해 매칭된 모든 부분을 추출
    const matches = inputString.match(specialCharPattern);

    if (matches) {
        // 배열을 문자열로 결합하여 반환
        return matches.join('');
    }

    return '';
}

// 사용 예제
console.log(extractSpecialCharacters("Date: 2021-07-09!")); // ":--!"
console.log(extractSpecialCharacters("Today's date is 07/09/21.")); // "'//."
console.log(extractSpecialCharacters("Date: 20210709!!")); // ":!!"
console.log(extractSpecialCharacters("Invalid date: $$07/##09//21..")); // ":$$//##//.."

```

**특정 문자의 위치 파악하기**

```jsx
function getYPositions(dateString) {
    const yPattern = /(?:y{2,}|[0-9]{2,})/g;
    const matches = [...dateString.matchAll(yPattern)];
    const positions = [];

    matches.forEach((match, index) => {
        if (match[0].length >= 2 && /^[0-9]+$/.test(match[0])) {
            const start = match.index;
            const end = start + match[0].length;
            for (let i = start; i < end; i++) {
                positions.push(i);
            }
        }
    });

    if (positions.length === 0) {
        throw new Error("지원되지 않는 날짜 형식입니다.");
    }

    return positions;
}

// 사용 예제
console.log(getYPositions("20210709")); // [0, 1, 2, 3] - yyyyMMdd
console.log(getYPositions("09072021")); // [4, 5, 6, 7] - ddMMyyyy
console.log(getYPositions("21-07-09")); // [0, 1] - yy-MM-dd
console.log(getYPositions("09-07-2021")); // [6, 7, 8, 9] - dd-MM-yyyy
console.log(getYPositions("abc2021xyz07gh09")); // [3, 4, 5, 6] - abc yyyy xyz07gh09

```
