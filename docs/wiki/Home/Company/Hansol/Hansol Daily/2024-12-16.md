## 법카 결재

Web 으로 된 SAP으로 결재 가능

- 다빈치 출장 여비 신청서 작성 후 캡쳐
- 엑셀 작성 후 캡쳐

해당 두 장을 합쳐서 두 번 업로드( 법카 증빙, 일비 증빙 따로 하기 위함)

web SAP

GL계정 - 6210285 ( 기존 연구소에서 변경)

코스트 센터 - 2287 ( 기존 연구소에서 변경)(25년 기준)

WBS 코드 - ITS-0061.001 (WBS 추가 작성란에 생성)
