# TypeScript 설치
![Screenshot-2023-05-17-at-7.52.05-AM.png](/img/이미지 창고/Screenshot-2023-05-17-at-7.52.05-AM.png)
## Command 설치
```bash
npm i typescript -g
```

## VsCode Plugin 설치
- Visual Studio 2017 / 2015 Update3 이후로는 기본으로 설치 되어있음

```bash
# 개발용으로 사용하는 typescript라는 것을 명시적으로 해주는 옵션
npm i typescript -D
npx tsc --init
```
